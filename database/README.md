# 🗄️ Banco de Dados

Esta pasta contém todos os arquivos relacionados ao banco de dados do projeto.

## Estrutura

```text
database/
├── scripts/
│   └── Scripts SQL da aplicação
│
├── backups/
│   └── Backups do banco de dados
```

## Tecnologias

* MySQL
* SQL
* MySQL Workbench

## Conteúdo

### Scripts

Contém os scripts responsáveis pela criação e manutenção do banco de dados.

### Backups

Contém cópias de segurança utilizadas durante o desenvolvimento do projeto.

> Novos scripts serão adicionados conforme a evolução da aplicação.
